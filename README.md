# City of Corpus Christi Water Media Dashboard

[![View Dashboard](https://img.shields.io/badge/Live-Dashboard-blue?style=for-the-badge&logo=googlecloud)](https://datastudio.google.com/reporting/e2cbc6a5-b89a-40e4-9cf0-b585e57b4797)

<h2>Overview</h2>
Built a real time dashboard aggregating published stories, media requests, and social user generated content across local, state, national, and international sources.

<h2>Data Engineering & Architecture</h2>
Engineered a Google BigQuery data warehouse via API integrations, automating data flow from Smartsheet through Google Sheets into BigQuery to power the dashboard and ad hoc reporting.

<h2>Impact & Results</h2>
The dashboard enabled leadership to quickly identify trends and allocate resources more effectively, resulting in faster response times and improved coordination across departments.
